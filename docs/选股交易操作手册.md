## 总纲:七条不可动摇的铁律

| # | 铁律 | 量化标准 |
|---|------|----------|
| 1 | **本金是子弹,风险永远优先** | 首次开仓 ≤ 总资金 10% 试错;单股 ≤ 20%-30%;同时持仓 3-5 只;不借钱、不用生活费 |
| 2 | **顺势而为,方向 > 技巧** | 月线看周期、周线看趋势、日线找机会、60 分钟找买卖点 |
| 3 | **概率思维,不预测** | 接受连亏 5 次仍理性;只看盈亏比 + 整体胜率;80% 时间在等待 |
| 4 | **纪律就是生命** | 设止损就止损,不拖延、不补仓、不频繁交易 |
| 5 | **慢即是快** | 趋势票拿大波段 > 频繁短线;上班族首选宽基 ETF 定投 |
| 6 | **知行合一** | 研究、调研、资金管理做透,行动才不犹豫 |
| 7 | **像保险公司一样投资** | 追求长期、稳定、可持续的确定性,不押单点 |

**期望值公式(决定一笔交易值不值得做):**
`EV = 胜率 × 单笔盈利 − (1 − 胜率) × 单笔亏损`,EV > 0 才执行。
盈亏比 3:1 时只需 25% 胜率即盈亏平衡;盈亏比 < 1 的交易坚决不做。

## 第一步:择时 —— 判断大盘环境(决定仓位上限)

> **核心认知:不要跳出大盘看板块和个股。** 大盘是风向标,逆水行舟必覆。

### 1.1 大盘趋势判定(四指标共振确认)

| 指标 | 多头(加仓) | 空头(降仓/空仓) |
|------|------------|------------------|
| **均线** | 指数站稳 60 日线,5>20>60 多头排列 | 跌破 60 日线,均线死叉空头排列 |
| **MACD** | 零轴上方金叉,红柱放大 | 零轴下方死叉,绿柱放大 |
| **量能** | 放量上涨,突破时量能放大 | 缩量阴跌,或高位放量滞涨 |
| **高低点** | 高点抬高 + 低点抬高(道氏) | 高点降低 + 低点降低 |

**确认信号**:指数上穿 60 日均线 + 成交量放大创新高/突破平台 + MACD 金叉柱状图翻红 → 三者共振 = 牛市确认。

### 1.2 流动性是牛熊根本

- **宽松期(放水)**:降准降息 + 成交量持续放大 → 容错率高,重仓成长/科技主线。
- **紧缩期(退潮)**:政策收紧 + 缩量 → 机会骤减,宁可少做不做。
- **判断双信号**:① 政策面(降准降息/国常会表态)是水龙头;② 成交量是水流的直接表现。"地量见地价,天量见天价"。

### 1.3 仓位上限映射(大盘决定总仓位)

| 大盘状态 | 总仓位 | 配置偏向 |
|----------|--------|----------|
| 牛市主升(多头共振) | **70%-80%** | 中长线龙头 50-60% + 波段 30-40% + 短线 10-20% |
| 震荡市(分歧轮动) | **50% 滚动** | 半仓滚动,30%-70% 区间高抛低吸 |
| 弱势/下跌(空头) | **0%-30%** | 防御为主,红利/高股息,或空仓等待 |

**牛市均线纪律(做长不做短):**
1. 不上 20 日均线,不买。2. 不破 5 日均线,不卖。3. 跌破 5 日线,减半仓。
4. 跌破 10 日线,清仓离场。5. 躺在 60 日均线以下的票,不要买。
6. 上升趋势中回调到 5 日均线是买点;回踩 20 日均线有支撑。

### 1.4 情绪周期(短线择时,6 阶段)

口诀:**冰点不动,启动试水,加速重仓,高潮减仓,退潮空仓。**

| 阶段 | 特征 | 操作 |
|------|------|------|
| 潜伏期(冰点末期) | 跌停/天地板频现 | 轻仓守底部趋势票,观察 |
| 启动期 | 题材萌芽,首板出现 | 小仓试水,敢打首板 |
| 加速期(最赚钱) | 2-3 连板,换手板频现 | 果断上车核心龙头(控仓防过热) |
| 情绪高潮 | 龙头 4-5 板,散户蜂拥,炸板增多 | 快进快出,**开始减仓** |
| 分化期(退潮) | 龙头见顶大阴杀跌带崩跟风 | **坚决离场空仓** |
| 冰点期 | 跌停频现,无人接力 | 轻仓观察,留意新题材萌芽 |

## 第二步:选股 —— 三周期共振选股法

> 核心口诀:**大周期定方向,中周期选结构,小周期抓节奏。三者共振出黄金点。**

### 2.1 第一层:大周期(月/周)定方向 —— 过滤器

**硬性条件(不满足直接淘汰):**
- 月线均线多头排列,或月线 MACD 金叉。
- 周线站稳 MA60,处于上升趋势。
- **新手极简系统**:日线所有均线在 250 日线之上且多头顺次排列 + 周线在 60 周线之上。

**回避**:熊市股、长期平台震荡股、下跌趋势股、ST/问题股。

### 2.2 第二层:中周期(日)选结构 —— 找形态

寻找以下回调结束 / 突破点(低吸区):
- 日线回踩 **20 日均线止跌回升**。
- W 底(双底)、头肩底、箱体震荡上沿突破、上升旗形。
- 平底结构(多次下探相似低点不破)+ 放量突破。
- 空中加油(强势股高位窄幅横盘整固后再次突破)。

### 2.3 第三层:小周期(60 分钟)抓时机 —— 确认入场

60 分钟级别出现以下信号为入场确认:
- 放量突破、缩量回踩企稳。
- MACD 金叉、底背离。
- KDJ 超跌金叉。

**三周期共振买入 = 成功率最高、风险可控。**

### 2.4 基本面叠加(趋势票必备)

- 行业景气度上行(处于产业周期成长期/政策驱动期)。
- 公司业绩持续增长(营收增速、毛利率、ROE 提升)。
- 政策利好、北上/机构资金持续流入。
- 产业资本增持/回购 = 最具参考意义的利好信号。

### 2.5 产业周期选行业(宏观面)

| 周期阶段 | 选股逻辑 | 配置 |
|----------|----------|------|
| **萌芽期** | 逻辑为王,小市值强逻辑(量子/6G) | 小仓位,严止损 |
| **成长期** | 业绩为王,龙头+技术优势二线(新能源/AI/半导体) | 重仓主线 |
| **成熟期** | 现金流为王,稳定分红低估值(白酒/电信/银行) | 防御配置 |
| **衰退期** | 通常不参与 | 极度低估可小仓左侧埋伏 |

**核心+卫星配置**:成长期龙头 + 成熟期稳定股占 70%;萌芽潜力 + 衰退博反转占 30%。

## 第三步:技术指标量化规则(选股引擎参数库)

### 3.1 均线系统(最核心,本书反复强调)

| 均线 | 作用 | 信号 |
|------|------|------|
| **MA5** | 短线生命线 | 主升浪沿 5 日线持股;收盘跌破减仓;回踩不破是买点 |
| **MA10** | 短线确认 | 5 日线上穿 10 日线 = 买点 |
| **MA20** | 机构生命线(中期核心) | 稳在上方 = 机构持仓;跌破不收回 = 趋势转弱 |
| **MA60** | 波段/中期趋势 | 站上 = 偏强;跌破 = 偏弱;突破 60/120 日线放量 = 趋势启动 |
| **MA120/250** | 长期趋势 | 半年线/年线,长期分水岭 |

**多头排列**:MA5 > MA10 > MA20 > MA60 → 强势多头,持有/做多。

**破五反五(洗盘信号)**:放量破 5 日线但快速反抽站上 = 主力洗盘,继续持有;缩量破后弱势回 5 日线 = 警惕假反抽。

### 3.2 520 战法(MA5 + MA20,波段核心策略)

| 信号 | 触发条件 | 操作 |
|------|----------|------|
| **金叉买入** | MA5 上穿 MA20 + 量温和放大 | 突破当天收盘前买入(放量可加仓) |
| **回踩买入** | 上涨中回调至 MA20 不破 + MA5 向上拐头 | 企稳后买入(放量更强) |
| **死叉卖出** | MA5 下穿 MA20 | 死叉当天/次日反弹无力时卖出 |
| **跌破卖出** | 股价跌破 MA20 且 MA5 跟随向下 | 跌破当天/次日未站回果断卖出 |

**止损**:跌破 MA20 或金叉后无力上攻跌回均线下方。

**提高胜率组合**:MACD 金叉 + 520 金叉 = 强买;KDJ 低位金叉 + 520 金叉 = 更稳买点。

### 3.3 趋势行情 5 日线交易法则(主升浪专用)

**适用**:明显上升趋势,已突破 60 日线,进入主升浪。

**买入(满足任意 3 个)**:
1. MA5 上穿 MA10、MA20,短期多头发散。
2. 放量上攻后首次回踩 MA5 不破。
3. 回踩缩量横盘,止跌信号(锤头线/阳包阴)。
4. 回踩 MA5 后次日收阳,量温和放大。
5. MACD 绿柱缩短或翻红。

**持股**:只要在 MA5 上方就持有;回踩能收回继续持有;长上影未破 MA5 可减仓不清仓。

**卖出**:
- 止盈:天量无法上攻/高位长上影;首次放量跌破 MA5(大阴线);MACD 顶背离;涨幅超 20%-30% 后剧烈波动。
- 止损:买入后连续 2-3 日无预期上涨;跌破 MA5 反弹不过再次回落;放量但价格反跌。

### 3.4 MACD 规则

| 信号 | 条件 | 操作 |
|------|------|------|
| **零轴上方金叉** | DIF、DEA 在零轴上方金叉 | **强势买入**(多头调整结束) |
| **零轴下方金叉** | 下跌趋势中金叉 | 规避(可能诱多) |
| **零轴上死叉** | 强势调整 | 多头中继,支撑不破可低吸 |
| **零轴下死叉** | 放量下跌 | **加速杀跌,不宜抄底** |
| **顶背离** | 股价新高,MACD 未新高 | 止盈/减仓,不追高 |
| **底背离** | 股价新低,MACD 未新低 | 关注止跌,不盲目杀跌 |

**柱状图**:放大=趋势延续;缩小=动能减弱可能转势;由负变正=反弹信号;由正变负=回调信号。

**背驰级别**:5 分/30 分/日线同时背驰 = 极佳转折点(重点交易信号)。

### 3.5 RSI 规则

| RSI | 状态 | 操作 |
|-----|------|------|
| > 80 | 极端超买 | 卖出 |
| 70-80 | 超买 | 考虑卖出 |
| 50 附近 | 整理 | 观望(RSI>50 多头可信,<50 空头可信) |
| 20-30 | 超卖 | 考虑买入 |
| < 20 | 极端超卖 | 买入 |

**组合强信号**:RSI 底背离 + MACD 金叉 = 强买入;RSI < 30 钝化后突破 30/50 = 反弹成立。

### 3.6 量价关系(量价四象限)

| 形态 | 含义 | 操作 |
|------|------|------|
| **价涨量增** | 健康上涨 | 买入/持有(放量突破关键位最佳) |
| **价跌量增** | 抛压加重 | 高位→卖出;末期→可能见底 |
| **价涨量缩** | 上涨乏力/诱多 | 规避/减仓 |
| **价跌量缩** | 抛压衰竭/筑底 | 末期→关注反弹 |

**特殊信号**:底部放量长阳=见底买入;天量天价=见顶卖出;放量突破=有效突破;放量滞涨=出货卖出。

### 3.7 换手率(识别主力行为)

| 换手率 | 含义 | 操作 |
|--------|------|------|
| 3%-5% | 主力吸筹(低位尤关注) | 关注/低吸 |
| 8%-15% | 主力资金大笔流入 | 跟进 |
| > 25% | 主力出货(高位+大阴线=见顶) | 卖出 |
| > 70% | 主力疯狂出逃 | 立即卖出 |

### 3.8 乖离率 BIAS(超买超卖量化)

| 条件 | 阈值 | 操作 |
|------|------|------|
| 个股超跌反弹 | BIAS6 < -6% | 技术性反弹买入 |
| 大盘/权重超跌 | BIAS6 < -4% | 买入 |
| 个股短线止盈 | BIAS6 > +6% | 获利回吐/卖出 |
| 波段减仓 | BIAS12/24 长期 +4%~+8% | 转头向下=波段调整,逢高减仓 |

### 3.9 筹码结构

| 筹码集中度 | 含义 | 操作 |
|------------|------|------|
| > 70% | 主力控盘强 | 重点关注(启动后易拉升) |
| < 40% | 筹码分散无控盘 | 规避 |

**筹码峰**:股价在峰上方=偏多(多数赚钱);在峰下方=偏空(套牢盘压力)。
**筹码集中 + MACD 金叉 = 启动信号。**

### 3.10 K 线形态信号

| 形态 | 位置 | 信号 |
|------|------|------|
| **平底结构** | 下跌末期 | 底部反转,放量突破=买入(右侧) |
| **仙人指路** | 低位/突破点 | 长上影阳线+放量,主力试盘,跟随做多 |
| **射击之星** | 高位 | 长上影小实体,涨势将尽,卖出 |
| **W 底/头肩底** | 底部 | 颈线突破回踩不破=买入 |
| **缺口(突破)** | 平台突破点 | 放量跳空突破不回补=顺势追涨 |
| **缺口(衰竭)** | 趋势末期 | 巨量跳空后迅速反转=止盈/反向 |

## 第四步:买入流程(三重确认 + 分批建仓)

### 4.1 永不被套战法(买入前三重确认,必须同时成立)

1. **形态成立**:趋势向上(均线多头)/ 平台放量突破 / 底部放量反弹。
2. **放量突破**:成交量明显放大,**最好高于前均量 1.5 倍**。
3. **趋势不破**:大周期方向向上,关键支撑未破。

### 4.2 三种买入法

| 方法 | 条件 | 适用 |
|------|------|------|
| **突破买入** | 放量突破压力位 + 缩量回踩确认 | 右侧交易,趋势启动 |
| **回调买入** | 回踩 20/60 日均线企稳低吸 | 左侧布局,降低成本 |
| **整理突破买入** | 平台缩量后放量突破站稳加仓 | 平台整理结束 |

### 4.3 分批建仓节奏

- **首次开仓 ≤ 10%** 试错。
- 确认趋势后**金字塔加仓**(底部重仓,越往上加越少)。
- 加仓只在**盈利基础上**,加仓资金必须设止损。
- **亏损股绝不再加仓**(错上加错);加仓 ≠ 补仓。

### 4.4 买入检查清单

- [ ] 大盘处于多头或震荡(非下跌趋势)?
- [ ] 大周期(周线)方向向上?
- [ ] 日线形态成立(突破/回调企稳/底部结构)?
- [ ] 60 分钟确认信号出现?
- [ ] 放量突破(量比前均量 ≥ 1.5 倍)?
- [ ] 盈亏比 ≥ 2:1(理想 3:1)?
- [ ] 止损位已明确(结构止损 / 1.5-2 倍 ATR)?
- [ ] 仓位符合风控(单笔风险 ≤ 账户 0.5%-1%)?

## 第五步:持仓与风控(保住本金 = 保住机会)

### 5.1 持股纪律(趋势不破不下车)

- 主升浪沿 MA5 持股;失守 MA5 部分止盈。
- 缩量调整无需恐慌,以 MA20/MA60 作防守位。
- **移动止盈**:每日上移止盈线参考 MA10 / ATR 下轨 / 前期支撑。
- **MACD 死叉 + 跌破 MA20 → 全清。**
- 突然放巨量破位 → 警惕撤退。

### 5.2 止损体系(三层)

| 类型 | 规则 | 阈值 |
|------|------|------|
| **硬止损** | 跌破前期支撑/MA20 | **5%-8%** 果断止损 |
| **软止损** | 均线死叉、放量下跌破位 | 信号触发即执行 |
| **时间止损** | 入场后无明显上涨 | **5-7 个交易日**(或一个月不涨即出) |

**短线止损 3%-5%,中长线 5%-10%。**

**本金亏损回本难度**:亏 10% 需涨 11.1%;亏 20% 需涨 25%;亏 50% 需涨 100%;亏 80% 需涨 400%。→ **保本比赚钱更重要。**

### 5.3 防守计划(账户级红线)

- 最大回撤容忍:**10%-15%**(触及全面降仓/停手复盘)。
- 单笔亏损 ≤ 账户净值 **0.5%-1%**。
- 单日亏损 ≥ **1.5%** → 当天强制停手;单周 ≥ **3%-5%** → 下周仓位减半。
- **头寸反推**:`头寸 = 允许亏损金额 ÷ (入场价 − 止损价)`。
- 组合分散:**单股 ≤ 20%,单行业 ≤ 40%**。

### 5.4 做T降成本(震荡市)

- **正T(顺势,低买高卖)**:下跌至支撑加仓 → 反弹卖出新增仓,仓位不变均价下降。
- **倒T(逆势,高卖低买)**:高位先卖出锁利 → 回调买回,降低风险。
- **新手不建议日内T**,先学波浪;做T用 **3 成仓或半仓滚动**,不满仓做T。

### 5.5 调仓纪律

- 每周/每月问:"如果今天空仓,我还会买它吗?"答否则调仓。
- 牛市不是用来解套的,是用来翻倍的;不耗死在不争气的票上。
- 减仓 ≠ 清仓(减仓最少保留 5 成);不要频繁换股。

## 第六步:卖出流程(逃顶 + 止盈止损)

### 6.1 卖点信号清单

| 类别 | 信号 | 操作 |
|------|------|------|
| **止盈** | 涨幅达 20%/50% 或前高 + 滞涨(连续上影/量缩) | 止盈 |
| **止盈** | 放量滞涨 + 次日无力新高 + 天量天价 | 卖出 |
| **止盈** | 加速上涨(涨停连板 + 角度>60° + 涨幅>50%) | 卖出 |
| **止损** | 跌破趋势线/MA20/MA60 + 反抽无力 | 止损 |
| **止损** | 高位放量大跌(跌幅>5%+放量) | 卖出(主力出货) |
| **止损** | MACD 高位死叉 + 顶背离 + 跌破 MA5 | 卖出 |
| **反转** | 跌破长期趋势线(半年线/年线) | 卖出 |
| **反转** | M 头/头肩顶 + 跌破颈线 + 放量加速下跌 | 卖出 |

### 6.2 3 天不创新高法则(量化卖点)

- **真正的强势不会有超过 3 天的回调**,否则持续震荡。
- **3 天不创新高 → 找卖点。**
- **强势板块见顶唯一原则:大阴线放量跌破 5 日线,或放量滞涨 → 坚决离场。**
- 跌下来**不要认为是机会去加仓!**

### 6.3 顶部信号总表(逃顶)

天量天价 / 放量滞涨 / 高位放量大阴线(断头铡刀)/ MACD 顶背离 / MACD 零轴下死叉 / RSI>80 回落 / 布林触上轨收缩 / 射击之星 / 黄昏之星 / 双顶(M头)/ 头肩顶 / 跌破 MA60 / 上涨角度>60° / 尾盘急拉>3% / 换手率>25%+高位大阴线 / 换手率>70%。

### 6.4 回调 vs 出货(决定加仓还是下车)

| 特征 | 回调(加仓) | 出货(下车) |
|------|------------|------------|
| 成交量 | **缩量**下跌 | **放量**下跌 |
| 支撑/反弹 | 有支撑,反弹迅速 | 反弹无力,连阴杀跌 |
| 技术面 | 未破位 | **破位 + 基本面拐点共振** |
| 资金流 | 主力未大规模流出 | 板块同步跳水,龙头带头砸盘 |

### 6.5 震仓 vs 出货(区分)

- **震仓**:打压后有承接,股价很快企稳甚至继续上涨;下跌**缩量**;不破重要支撑。
- **出货**:放量下跌后无人承接,股价持续走低;放**巨量**价格无法回升。
- **关键看:打压后有没有资金护盘。**

## 交易纪律速查卡(打印贴在屏幕旁)

### 买入前必问
1. 大盘环境允许吗?(不满仓逆势)
2. 三周期共振了吗?
3. 盈亏比 ≥ 2:1 吗?
4. 止损位设好了吗?
5. 仓位符合风控吗?(单笔风险 ≤ 1%)

### 持仓中必守
- 趋势不破不下车(MA5/MA20 防守)。
- 缩量调整不恐慌,放量破位必撤退。
- 加仓只在盈利基础上,亏损绝不补仓。
- 移动止盈线每日上移。

### 卖出时必决
- 3 天不创新高 → 找卖点。
- 强势股跌破 5 日线放量 → 坚决离场。
- 放量滞涨 / 天量天价 → 止盈。
- 达到目标位 → 分批落袋,不贪最高点。

### 绝对禁忌
1. **连续大阳不买**(末期出货)。
2. **连续大阴不卖**(主力逼筹码)。
3. **不满仓、不重仓单一标的、不借钱**。
4. **不追高**(涨停板/连续大阳后不买)。
5. **亏损股绝不补仓摊平**。
6. **消息反着听**(盘口/股吧/媒体)。

## 附录:核心战法速查

### 龙头战法(情绪票,短线)
- 口诀:**一板观察、二板跟踪、三板确认**。
- 抓首板(换手>10%+封单大+快速封板)→ 追二板(迅速封住)→ 三板定性(缩量封板/抱团封死=核心龙头)→ 四五板(情绪高潮,准备收割)。
- **不抄底、不赌反包、不抢反弹。龙头是走出来的,不是看出来的。**

### 定投(指数 ETF,长线)
- 底仓:沪深 300 / 中证 A500 / 科创 ETF。
- 节奏:大跌大买、小跌小买,每月固定时间/金额。
- 周期:持有 3-5 年以上;目标 30% 分批止盈。
- 微笑曲线:跌时积累便宜筹码,涨时享受收益。

### 抄底(盈亏比管理)
- 不是预测最低点,是风险收益分析:下行空间有限 + 上行空间可观。
- 趋势性下跌中不急于抄底;等企稳信号(缩量+支撑位+反转形态)。
- **分批建仓**,不一次性砸进去。

### 季节性板块轮动(月度路线)
1月消费 / 2月农业(一号文件) / 3月大会 / 4月季报 / 5月抱团 / 6月中小盘 / 7月水利 / 8月消费新品 / 9月军工文旅 / 10月物流电商 / 11月燃气 / 12月绩优。

## 结语:投资即修心

- **知常曰明**:看清周期与人性,盛极必衰,否极泰来。
- **反者道之动**:上涨孕育下跌,下跌积蓄反转。
- **无为而治**:不妄动、不强求,守静待时,顺势而为。
- **破相**:股价≠价值,交易≠投资,信息≠洞察,亏损≠失败。
- 市场不是用来预测的,而是用来映照你的认知结构。从**看行情 → 看结构 → 看自己**。

> 唯一不变的,就是变化本身。在顺应变化中寻找规律,在不确定中保持清醒,这才是穿越周期、立于不败之地的根本。
