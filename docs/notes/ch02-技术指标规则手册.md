## 1. 量价关系(量价时空、量价配合、量价背离)

### 1.1 量价时空四要素
- **量(成交量)**:市场活跃度,领先于价格变化("量先行,价跟随")。
- **价(价格)**:供需平衡结果。趋势线/支撑阻力/K线形态判断。
- **时(时间)**:短期(几天-几周,短线)、中期(几周-几个月,波段)、长期(几个月-几年,趋势)。
- **空(空间)**:目标区域,通过历史高低点、斐波那契回调、黄金分割、形态(三角形/头肩底)测算。

### 1.2 量价四象限规则
| 形态 | 含义 | 触发条件 | 操作 |
|---|---|---|---|
| **价涨量增** | 健康上涨/趋势强化 | 价格上涨 + 成交量持续放大 | **买入/持有**(放量突破关键位尤其有效) |
| **价跌量增** | 抛压加重/恐慌 | 价格下跌 + 成交量放大 | 下跌初期→可能探底;**高位→卖出**;下跌末期→可能见底 |
| **价涨量缩** | 上涨乏力/诱多 | 价格上涨 + 成交量萎缩 | **规避/减仓**(谨防假突破、拉高出货) |
| **价跌量缩** | 抛压衰竭/筑底 | 价格下跌 + 成交量萎缩 | 下跌末期→**关注反弹**;可能筑底信号 |

### 1.3 特殊量价信号
- **底部放量(长阳)**:见底信号,抄底资金入场 → **买入**。
- **天量天价**:见顶信号,主力派发 → **卖出**(高位放量大阴线/"断头铡刀")。
- **缩量横盘**:在重要支撑位→蓄势待发;在阻力位→警惕假突破。
- **放量突破**:突破压力位时量能放大 → **有效突破**,趋势启动。
- **无量上涨**:上涨乏力,可能假涨/诱多 → **谨慎看多**。
- **无量下跌**:可能是洗盘/震仓,下跌幅度有限 → **谨慎看空**。
- **放量滞涨**:主力出货 → **卖出**。
- **核心原则**:量是水,价是船,水涨船高;突破关键承压位必须有量。

## 2. 量比、换手率、委比

### 2.1 量比(市场活跃度)
> 量比 = 当日开市后平均每分钟成交量 / 过去5个交易日平均每分钟成交量

- 量比低 → 市场观望;量比高 → 市场活跃;极端量比 → 谨慎操作。
- 量比本身非买卖信号,需结合趋势、换手率、K线形态。

### 2.2 看盘时间点量比信号
| 时间点 | 观察项 | 阈值/信号 | 操作 |
|---|---|---|---|
| 早上 9:45 | 量比 | 量比 > 5 | **警惕主力出货** |
| 下午 1:15 | 拉升量 | 首波拉升不带量 | **必回落**(规避追高) |
| 下午 2:50 | 尾盘急拉 | 尾盘急拉 > 3% | 次日低开概率高,**谨慎** |

### 2.3 换手率阈值(主力行为识别)
| 换手率区间 | 含义 | 操作 |
|---|---|---|
| 3% ~ 5% | 主力吸筹(低位尤其关注) | **关注/低吸** |
| 8% ~ 15% | 主力资金大笔流入,股价水涨船高 | **跟进** |
| > 25% | 主力出货(高位 + 大阴线 = 见顶) | **卖出** |
| > 70% | 主力疯狂出逃,未来大跌概率高 | **立即卖出,不抱侥幸** |
| 新股开板 > 50% 且维持 20% 左右 | 后市仍可上涨 | **关注** |

### 2.4 委比(买卖盘力量对比)
> 委比 = (委买手数 - 委卖手数) / (委买手数 + 委卖手数) × 100%

| 委比区间 | 含义 | 配合条件 | 操作 |
|---|---|---|---|
| **> 0** | 买盘强于卖盘 | - | 偏多 |
| **> +50%** | 大量买盘支撑 | + 股价上涨 | **买入/看涨**(主力吸筹/趋势启动) |
| **+50% 以上但股价未突破压力** | 诱多陷阱 | 委比维持 +60% 反复冲高不破 | **规避**(假突破) |
| **< 0** | 卖盘强于买盘 | - | 偏空 |
| **< -50%** | 抛压沉重 | + 股价下跌 | **卖出/看跌**(主力出货/恐慌) |
| **负值(如 -40%)但股价低位止跌** | 筑底信号 | 股价未创新低 | **关注吸筹**(主力低位吸筹) |
| **= 0** | 买卖平衡 | - | 观望,可能盘整或方向选择 |

**委比组合信号**:
- 委比 > 0 + 量比扩大 → 买方力量增加,**短期看涨**。
- 委比 > 0 + 换手率高 → 新资金介入。
- 委比为正 + 长阳线 → 主力拉升。
- 委比为负 + 长阴线 → 主力出货。

## 3. MACD 指标

### 3.1 参数与组成
- **DIF(快线)** = 短期 EMA(12) - 长期 EMA(26)
- **DEA(慢线)** = DIF 的 9 日 EMA
- **MACD 柱** = DIF - DEA(柱状图,反映多空力量)

### 3.2 买卖信号规则
| 信号 | 触发条件 | 趋势环境 | 操作 |
|---|---|---|---|
| **金叉(买)** | DIF 上穿 DEA | 上涨趋势 | **买入**(成功率较高) |
| **金叉(诱多)** | DIF 上穿 DEA | 下跌趋势 | **规避**(可能诱多) |
| **死叉(卖)** | DIF 下穿 DEA | 下跌趋势 | **卖出/离场** |
| **零轴上方金叉** | DIF、DEA 在零轴上方金叉 | 强势市场 | **强势买入**(多头调整结束) |
| **零轴下方死叉** | DIF、DEA 在零轴下方死叉 | 弱势市场 | **危险/加速下跌** |
| **零轴附近金叉/死叉** | 接近零轴 | 震荡 | 信号弱,**易反复,谨慎** |
| **零轴上死叉** | 死叉但 MACD 长期在零轴上方 | 强势调整 | 视为**多头中继**,支撑位不破可低吸,勿过早做空 |
| **零轴下死叉** | 死叉且 MACD 在零轴下方且放量下跌 | 弱势延续 | **加速杀跌前兆,不宜抄底**,逢高减仓/顺势做空 |

### 3.3 柱状图规则
- 柱状图放大 → 多空力量增强,趋势延续。
- 柱状图缩小 → 动能减弱,可能转势。
- 柱状图由负变正 → 可能反弹信号。
- 柱状图由正变负 → 可能回调信号。

### 3.4 背离(趋势反转)
- **顶背离(卖)**:股价新高,但 MACD 未新高 → 上涨乏力,**止盈/减仓,不追高**。
- **底背离(买)**:股价新低,但 MACD 未新低 → 下跌动能衰竭,**关注止跌信号,不盲目杀跌**。
- **背驰 vs 背离**:背离看 DIF 值(一个点);背驰看红绿柱面积变化。

### 3.5 背驰级别(决定调整周期)
| 级别 | 周期 | 调整时长 | 操作 |
|---|---|---|---|
| 小级别 | 1分/5分 K线 | 短期调整 | 日内 T+0 减仓 |
| 中级别 | 15分/30分 | 3-5 个交易日波段调整 | 波段卖点 |
| 大级别 | 日线/周线 | 数周至数月主跌浪 | **减仓离场/反向布局** |
| **多级别共振** | 5分+30分+日线同时背驰 | 极佳转折点 | **重点交易信号** |

### 3.6 实战要点
- 强势趋势中:金叉后回踩不破 DEA + 柱状图由负转正 = 较好买点。
- 震荡市:MACD 频繁金叉死叉,需结合均线、成交量。
- 结合指标:MACD + 均线、KDJ、成交量;突破支撑/压力位 + 柱状图放量更可靠。

## 4. RSI 指标

### 4.1 超买超卖阈值
| RSI 值 | 状态 | 操作 |
|---|---|---|
| **> 80** | 极端超买 | **卖出**(更易回调) |
| **70 ~ 80** | 超买 | **考虑卖出**(价格可能回调) |
| **50 附近震荡** | 整理/方向不明 | 观望 |
| **30 ~ 50** | 偏空 | 谨慎 |
| **20 ~ 30** | 超卖 | **考虑买入**(价格可能反弹) |
| **< 20** | 极端超卖 | **买入**(更易反弹) |

> 注意:超买超卖适用于震荡行情;单边趋势中可能长时间超买/超卖,不宜盲目进场。

### 4.2 中轴(50)趋势判断
- RSI > 50 → 多头趋势,**买入信号更可信**。
- RSI < 50 → 空头趋势,**卖出信号更可信**。
- 上升趋势中 RSI 回踩 50 附近反弹 → **加仓机会**。
- 下降趋势中 RSI 反弹至 50 附近受阻 → **可能继续下跌**。

### 4.3 金叉/死叉
- RSI 金叉(买):短周期 RSI 上穿长周期 RSI;配合股价放量突破更强。
- RSI 死叉(卖):短周期 RSI 下穿长周期 RSI;配合股价跌破重要支撑更可靠。

### 4.4 背离
- **顶背离(卖)**:股价新高,RSI 未新高 → **减仓,不追高**。
- **底背离(买)**:股价新低,RSI 未新低 → **关注企稳,适当低吸**。

### 4.5 RSI 组合信号(提高准确率)
- RSI 底背离 + MACD 金叉 → **强买入**。
- RSI 超买但股价在强势均线(10日线)上 → 可继续持有。
- RSI 超卖 + 放量长下影线 → **反弹信号**。
- RSI < 30 钝化后向上突破 30/50 → **反弹成立(买)**。

## 5. 均线系统

### 5.1 各均线用法
| 均线 | 类型 | 用法 | 操作 |
|---|---|---|---|
| **5日线** | 短期/短线 | 上穿=短线买入;下穿=短线卖出 | 短线交易主参考 |
| **10日线** | 短期/短线 | 同上,稍稳定 | 短线交易 |
| **20日线** | 中短期/"机构生命线" | 稳在上方=机构持仓/加仓;跌破且不收回=趋势转弱(机构减仓/止盈) | 中期趋势核心线 |
| **30日/60日线** | 中期/波段 | 比短期更稳定,信号更可靠 | 波段操作 |
| **120日线(半年线)/200日线(年线)** | 长期 | 长期站上=市场偏强;跌破=市场偏弱 | 长期趋势判断 |

### 5.2 交叉信号
- **黄金交叉(买)**:短期均线上穿长期均线。
- **死亡交叉(卖)**:短期均线下穿长期均线。

### 5.3 破五反五(5日线洗盘信号)
- **破五**:股价跌破 5 日均线(短线看似走弱)。
- **反五**:很快(当日或次日)重新站上 5 日均线。
- **判定**:
  - 放量破五但快速反抽站上 → 主力洗盘,**继续持有/反包放量时小仓跟进**。
  - 缩量破五后反弹回 5 日线 → 回踩确认支撑,**趋势继续**。
  - 反抽必须有效(收盘收回 5 日线 + 量能配合)。
  - 弱势缩量回 5 日线易再跌 → **警惕假反抽**。
- 口诀:破五不怕,反包才大;破而不立,小心再杀。

### 5.4 多头排列
- 短期均线在上、长期均线在下,依次排列(如 5 > 10 > 20 > 60) → **强势多头,持有/做多**。

### 5.5 斜率
> 斜率 = 价格变化 ÷ 时间变化(涨跌快慢与方向)
- 斜率大且向上 → 涨得猛,动能强(但越陡风险越高)。
- 斜率大且向下 → 跌得快,抛压重(所有反弹必须跑路,趋势会延续)。
- 斜率平缓 → 走势平稳,行情震荡。
- **只有斜率趋缓横盘震荡,才有再次发动行情的机会**。

### 5.6 乖离率 BIAS 阈值
> BIAS = (收盘价 - 移动平均价) / 移动平均价 × 100%

#### 买入参考
| 条件 | 阈值 | 含义 | 操作 |
|---|---|---|---|
| 短线超跌反弹(个股) | **BIAS6 < -6%** | 短期下跌过度 | **技术性反弹买入**(配合支撑位、缩量下跌) |
| 短线超跌反弹(大盘/权重股) | **BIAS6 < -4%** | 短期下跌过度 | **买入** |
| 波段低吸 | **BIAS12、BIAS24 同处 -4% ~ -6%** 且止跌 | 中期趋势未坏 | **分批建仓** |

#### 卖出参考
| 条件 | 阈值 | 含义 | 操作 |
|---|---|---|---|
| 短线止盈(个股) | **BIAS6 > +6%** | 短期涨幅过快 | **获利回吐/卖出**(放量滞涨风险更大) |
| 短线止盈(大盘/权重股) | **BIAS6 > +4%** | 短期涨幅过快 | **卖出** |
| 波段减仓 | **BIAS12、BIAS24 长期处 +4% ~ +8%** | 持续脱离均线 | 一旦转头向下=波段调整开始 → **逢高减仓** |

#### 关键提醒
- 买入不可单看 BIAS,需结合支撑位、量能、形态,避免"越跌越买"陷阱。
- 卖出不宜过早:强趋势(主升浪)中 BIAS 可能高位钝化,**分批减仓**而非一次清仓。
- BIAS 适合节奏把握,不适合单独作为买卖依据。

### 5.7 济安线(补充)
- 持币观望:济安线蓝色 + 辅助线绿色 → **观望,避免进场**。
- 买入:济安线走平 + 辅助线上穿济安线 + 济安线变粉红/辅助线变黄 → **买入**。
- 卖出:辅助线下穿济安线(死叉)+ 济安线变蓝 → **卖出**。
- 参数:默认 30,经验值调为 60。适用于趋势行情,横盘整理指导性弱。

## 6. 支撑位与压力位

### 6.1 如何找支撑位
1. **均线支撑**:20日、30日、60日均线(股价接近 + 止跌信号)。
2. **前期低点**:历史明显低点或震荡区域底部;越多次触及不破,支撑越强。
3. **趋势线**:连接多个低点的上升趋势线。
4. **成交密集区(筹码峰)**:成交量放大的震荡区。
5. **技术指标辅助**:布林带下轨、MACD 背离、RSI 超卖。

### 6.2 压力位操作
- **高抛减仓/止盈**:股价逼近压力位 + 见顶信号(放量滞涨、长上影线、K线反转)→ **减仓/清仓**。
- **等待突破回踩再做多**:判断可能突破 → 等突破确认(放量/收盘突破)→ 回踩不破(压力变支撑)→ **回踩做多**。

### 6.3 支撑变压力(经典转换)
- **原理**:跌破支撑后,原支撑位变为反弹压力(套牢盘解套抛售 + 趋势交易者加仓空单)。
- **判定**:跌破支撑 → 持续在下方横盘 → 反弹至原支撑区受压 = 压力位成立。
- **再转换**:若放量有效突破该压力位 → 压力再次变支撑。

### 6.4 回踩与破位判定
| 类型 | 特征 | 操作 |
|---|---|---|
| **回踩确认(买)** | 突破后短期回落测试支撑;回踩不破 + 缩量回踩放量反弹;均线多头排列保持;MACD/RSI 未明显走弱 | **低吸买入** |
| **破位下跌(卖)** | 跌破支撑且不快速收回;放量下跌(恐慌出逃);均线死叉/MACD绿柱放大/RSI超卖;反抽无力 | **卖出/止损** |

### 6.5 突破回踩 vs 做底回踩
- **突破回踩**:上涨中突破阻力位后回踩;回踩不破 + 缩量回踩 + 带量反弹 → **确认支撑,继续上涨**。
- **做底回踩**:下跌后震荡筑底;多次回踩不破 + 放量企稳 + 短期均线(5/10日)走平拐头 → **筑底成功,准备反弹**。

### 6.6 常见误区
- ❌ 追高在压力位附近买入。
- ❌ 单凭一次冲高认定突破(需"放量 + 站稳")。
- ❌ 忽视大盘环境与基本面。

## 7. 筹码结构

### 7.1 筹码集中度阈值
| 筹码集中度 | 含义 | 操作 |
|---|---|---|
| **> 70%** | 筹码很集中,主力控盘强 | **重点关注**(启动后易拉升,少人砸盘) |
| **< 40%** | 筹码分散,无控盘 | **规避**(易震荡、无方向) |

### 7.2 筹码峰判定
- **筹码峰**:筹码分布图中成交量最多、最密集的价格区域(主力建仓区或散户套牢区)。
- **位置判断**:
  - 股价在筹码峰**上方** → 多数人赚钱,情绪稳定,有支撑 → **偏多**。
  - 股价在筹码峰**下方** → 多数人套牢,涨至峰附近抛压大 → **偏空/压力**。
- **密集程度**:
  - 集中("一个大山头")→ 主力控盘,走势干净。
  - 分散("好几个小山头")→ 市场混乱,涨跌乏力。
- **筹码峰移动**:
  - 下移 → 主力压盘吸筹。
  - 上移 → 主力控盘上涨。
  - 不动 + 价格震荡 → 洗盘或诱多诱空。

### 7.3 筹码结构三要点
1. 集中 vs 分散(主力 vs 散户)。
2. 筹码在高 vs 低(压力 vs 支撑)。
3. 筹码动不动(吸筹、洗盘、拉升、出货)。

### 7.4 筹码配合指标
- 筹码 + 成交量:看量价配合。
- 筹码 + 均线:筹码集中在 20 日线上方 → 强势整理。
- 筹码 + K线:筹码下方有阳线平台 → 易形成支撑。
- 筹码集中 + MACD 金叉 → **启动信号**。
- 筹码集中 + 放量 → 主力正在运作;集中但缩量 → 隐藏建仓,未到拉升。

## 8. K线形态

### 8.1 平底结构
- **形态**:一段时间内多次下探到相似低点但未跌破,形成水平支撑区域。
- **特征**:多个低点处于相似价位(误差不大);每次回落止步于相近低点;时间跨度几天至几周;成交量逐渐缩小或突破时放大。
- **操作**:**底部反转信号**,常见于下跌趋势末期;放量突破阻力位 → **买入(右侧交易)**。
- **止损**:跌破平底 → 支撑失效,**技术破位需止损**。
- **配合**:MACD 底背离、量能分析综合判断。

### 8.2 孕线
- **形态**:两根 K 线;第一根(母线)实体较大;第二根(子线)实体较小,最高最低价被母线包裹。
- **市场含义**:多空力量暂时均衡,趋势犹豫或潜在转折。
  - 高位孕线(尤其孕阴线)→ 上涨动能衰竭 → **谨慎追涨,可能调整**。
  - 低位孕线(尤其孕阳线)→ 空头力量减弱 → **可能反弹**。
- **确认**:
  - 孕线后向上突破母线高点 → **看涨**。
  - 孕线后向下跌破母线低点 → **看跌**。
- 单独信号弱,需结合成交量、均线、MACD。

### 8.3 仙人指路(看涨)
- **形态**:一根带长上影线的阳线,收盘价偏高,成交量突然放大。
- **位置**:低位、突破点、横盘末尾。
- **后续**:1-2 天可能小幅回调或缩量横盘,然后再次放量突破。
- **操作**:**主力试盘/试拉信号**,**跟随做多**(缩量不跌确认)。

### 8.4 射击之星(看跌)
- **形态**:上影线很长,实体很小(像挂在空中的星星);位于高位或重要压力区。
- **后续**:第二天跳空低开或放量下跌 → 确认短期顶部。
- **操作**:**冲高受阻,涨势将尽**,**卖出/规避**。

### 8.5 仙人指路 vs 射击之星区分
- 低位/突破点/横盘末尾 + 长上影阳线 + 后续缩量不跌 → **仙人指路(看多)**。
- 高位/大涨之后 + 长上影小实体 + 后续放量回落跌破最低点 → **射击之星(看空)**。

### 8.6 顶部/底部 K 线组合
- **顶部信号**:射击之星、黄昏之星、吞没阴线、跳空高开回落。
- **底部信号**:锤头线、晨星形态、吞没阳线、跳空低开回升、十字星、阳包阴、锤子线。

### 8.7 缺口理论
| 缺口类型 | 位置 | 特征 | 操作 |
|---|---|---|---|
| **突破缺口** | 整理平台/重要形态突破点 | 成交量放大、价格跳空突破支撑/阻力;不易回补 | **顺势参与/追涨杀跌** |
| **运行/流通缺口(中继)** | 趋势进行中间 | 表明趋势强劲、资金持续进场;一般不补 | **持股/持币待涨**;可作量度升幅测量 |
| **消耗/衰竭缺口** | 趋势末期(顶/底) | 成交量巨大、跳空幅度大,之后迅速反转;主力出货 | **警惕诱多/诱空**,止盈或反向操作 |
| **普通缺口** | 震荡盘整期间 | 无明显量能配合,易很快回补 | **不宜参与**,仅作短线参考 |

#### 缺口实战用法
- **趋势强弱**:缺口向上不回补 → 多头强;快速回补 → 多头后劲不足。
- **买卖时机**:突破缺口后回踩缺口边缘不破 → **加仓**;衰竭缺口后见放量大阴线 → **卖出**。
- **目标位(流通缺口量度升幅)**:(起涨点到缺口距离) × 2 = 预期涨幅。
- **形态配合**:平台突破 + 缺口信号更强;放量缺口信号强,缩量易回补。

### 8.8 涨幅与振幅
- **振幅 < 涨幅**:单边上涨,回调少,抛压轻 → **强势延续信号**(上升趋势中尤甚)。
- **振幅 > 涨幅**:冲高回落或震荡上行,多空分歧大 → 高位需警惕上涨乏力/诱多。

## 9. 顶底拐点与中枢

### 9.1 顶底拐点定义
- **顶部拐点**:上涨趋势转下跌趋势的转折点 → **卖出时机**。
- **底部拐点**:下跌趋势转上涨趋势的转折点 → **买入时机**。

### 9.2 经典工具判断法
1. **趋势线与通道**:上升趋势线被有效跌破 → 顶部拐点;下降趋势线被有效突破 → 底部拐点。
2. **K线组合**:见第 8.6 节。
3. **MACD**:死叉 → 顶部拐点;金叉 → 底部拐点;顶/底背离预警。
4. **RSI**:RSI > 80 且开始回落 → 超买,顶部临近;RSI < 20 且开始上升 → 超卖,底部可能到来。
5. **布林带**:触及上轨并收缩 → 顶部可能;跌破下轨并收回 → 底部可能;收口 + 突破 → 趋势拐点。
6. **成交量**:顶部=放量滞涨、天量天价;底部=缩量止跌、地量地价、放量反弹。

### 9.3 相对顶与相对底(中枢博弈)
- **中枢**:主力与市场形成的平衡区(博弈核心区),多空力量暂时均衡。
- **相对顶**:价格触及中枢上沿 → 多方阶段性透支、筹码松动。
- **相对底**:价格回落至中枢下沿 → 空方能量衰竭、承接资金重现。
- **意义**:顶是强势中的休息,底是下跌中的蓄势;围绕中枢一涨一跌寻找新平衡。

### 9.4 实战判断技巧
- **多重验证**:形态 + 指标 + 成交量综合判断,不靠单一信号。
- **时间周期配合**:日线→短期拐点;周线→中期拐点;月线→长期趋势反转。
- **趋势确认再操作**:不"猜底/赌顶",等形态完成 + 有效突破/跌破趋势线 + 放量确认。
- **常见误区**:轻信单一指标、忽略大盘趋势、不设止损止盈。

### 9.5 空中加油(中继形态)
- **形态**:强势上涨后不深度回调,高位横盘/窄幅震荡整固,随后再次突破上行。
- **判定**:整理期间无明显放量出货;主力控盘蓄势;未破位下行;类似旗形/上升三角形。
- **操作**:**强势股续涨信号**,可作加仓点/二次买入点(比第一波启动风险更低)。等突破确认 + 量能配合入场。
- **风险**:伪空中加油(整理后放量下跌)= 出货;判断标准=能否有效突破横盘区域上沿 + 量能配合。
- **止损**:整理区下轨被有效跌破 → 及时止损。

## 10. 缠论三类买卖点

### 10.1 第一类买点(趋势回调背驰点)
- **位置**:上涨趋势中首次回调形成的低点,且该回调出现**背驰**。
- **特征**:回调力度减弱,动能背离,预示可能止跌反弹。
- **本质**:下跌趋势走完,出现新上涨中枢后,在背驰确认末端形成的第一个买点。
- **操作**:**买入**(背驰确认)。

### 10.2 第二类买点(中枢震荡后突破点)
- **位置**:价格在中枢区域震荡后,向上突破中枢上沿的位置。
- **特征**:趋势确认,突破中枢,预示新一轮上涨开始。
- **本质**:上涨趋势回调,价格回到中枢或中枢附近,再次向上离开时。
- **操作**:**买入/加仓**(突破中枢上沿)。

### 10.3 第三类买点(趋势延续回调确认点)
- **位置**:第二类买点后上涨过程中,小幅回调后再次上攻的位置;新中枢突破时。
- **特征**:回调幅度小,趋势延续,适合加仓或跟进。
- **本质**:**趋势确认 + 中枢抬高 + 新中枢突破**。至少两个上涨中枢已形成且走势上涨;新中枢高于旧中枢;价格从新中枢向上离开。
- **操作**:**最安全买点**,适合中长线;突破新中枢后第一笔上涨确认时买入。
- **止损**:价格重新跌回新中枢内部 → 第三类买点失败,**及时止损**。

### 10.4 缠论买卖点误区
- 没有新中枢就找第三类买点 → 不成立(需至少两个连续上涨中枢)。
- 新中枢未高于旧中枢 → 不成立。
- 混淆二买(中枢回踩确认)与三买(新中枢突破)。
- 级别要一致(明确 30 分钟/日线等级别)。

### 10.5 缠论背驰找卖点(不同级别)
- 见第 3.5 节 MACD 背驰级别表。小级别背驰只适合短线高抛,大级别背驰需减仓离场/反向布局。

## 11. 个股买点与卖点汇总

### 11.1 买点信号清单
| 类别 | 买点 | 触发条件 | 操作 |
|---|---|---|---|
| **趋势** | 上涨趋势回调买点 | 长期均线向上 + 回调至 20日均线/趋势线/中枢下沿 + 回调缩量企稳放量反弹 + K线止跌(十字星/阳包阴/锤子线) + MACD金叉/RSI超卖反弹 | **低吸买入** |
| **趋势** | 强势股回踩 | 回踩 5日/10日均线企稳 | **短线买入** |
| **趋势** | 波段股回调 | 回调至 20日/60日均线不破 | **中期买入** |
| **突破** | 突破买点 | 突破前期高点/箱体上沿 + 放量突破 + 回踩确认不破 + MACD金叉/KDJ向上/布林带上轨打开 | **追涨买入**(无量/快速跌回则止损) |
| **形态** | W底(双底) | 两次探底,第二次不破前低 + 颈线突破回踩不破 + 第二底放量 + MACD底背离金叉 | **买入** |
| **形态** | 头肩底 | 左肩-头-右肩结构 + 右肩缩量 + 颈线突破放量拉升 + 回踩不破 | **买入** |
| **形态** | 三角形突破 | 收敛整理 + 突破上轨放量 + 突破当日/次日不破突破点 | **短线买入** |
| **量价** | 缩量回调+放量反弹 | 回调缩量(主力未出货)+ 反弹放量突破短期压力 | **低吸买入** |
| **量价** | 放量突破+回踩确认 | 放量突破箱体上沿/重要均线 + 回踩缩量确认支撑 + 拉升 | **短线买入** |
| **指标** | MACD金叉 | DIF 上穿 DEA;0轴下方=超跌反弹;0轴上方=趋势延续 | **买入** |
| **指标** | RSI低位钝化+回升 | RSI < 30 超卖 + 向上突破 30/50 | **超跌反弹买入** |
| **指标** | 布林带下轨反弹 | 股价触及下轨 + 止跌信号 + 收回中轨 | **回调买入** |
| **缠论** | 一/二/三买点 | 见第 10 节 | **买入** |

#### 买点风控
- **止损**:跌破趋势线/形态颈线/箱体底部/突破失败回落原区间 → 果断止损。
- **资金管理**:短线仓位轻,快进快出;趋势交易分批建仓。
- **多信号结合**:不单看一个信号。

### 11.2 卖点信号清单
| 类别 | 卖点 | 触发条件 | 操作 |
|---|---|---|---|
| **止盈** | 目标位卖出 | 涨幅达 20%/50% 或技术目标位(前高) + 滞涨(连续上影线/量能缩小) + 偏离均线过远 + 压力位缩量滞涨 | **止盈** |
| **止盈** | 放量滞涨卖出 | 大涨但成交量极度放大 + 次日无力新高 + 天量天价 + 三连阳放量滞涨 | **卖出** |
| **止盈** | 加速上涨后卖出 | 连续涨停/大阳线 + 上涨角度 > 60° + 短期涨幅 > 50% + "高台跳水"信号 | **卖出** |
| **止损** | 跌破重要支撑 | 跌破趋势线/20日/60日均线 + 反抽无力 + 跌破前低/箱体底部 | **止损卖出** |
| **止损** | 放量下跌卖出 | 高位放量大跌(跌幅 > 5% + 放量) + 跌破支撑 + 放量加速下跌 | **卖出(主力出货)** |
| **止损** | MACD死叉卖出 | MACD高位死叉 + 顶背离 + 死叉后跌破 5日均线 + 反弹无力 | **卖出** |
| **趋势反转** | 破位卖出 | 跌破上升趋势线/均线系统 + 跌破 60日均线反抽无力 + 跌破长期趋势线(半年线/年线) | **卖出** |
| **趋势反转** | 双顶/头肩顶 | M头/头肩顶形成 + 跌破颈线 + 放量加速下跌 | **卖出** |
| **趋势反转** | 高位放量阴线 | 创新高后次日大阴线吞没前阳 + 放量下跌 + 跌幅 > 5% | **卖出** |

#### 卖点风格要点
- **短线**:关注放量滞涨、技术形态破位。
- **波段**:止盈目标位卖出,或趋势转弱时卖出。
- **趋势**:趋势破坏(跌破关键均线/趋势线)时卖出。
- **所有风格**:严格止损,避免亏损扩大。

### 11.3 回调 vs 出货判定(决定加仓还是下车)
| 特征 | 回调(健康休息) | 出货(主力撤退) |
|---|---|---|
| 成交量 | **缩量**下跌 | **放量**下跌 |
| 支撑/反弹 | 有明显支撑位,反弹迅速 | 反弹无力,连阴杀跌 |
| 板块/指数 | 无系统性利空 | 利好出尽,消息面转弱,行业见顶 |
| 资金流 | 主力未大规模流出 | 板块同步跳水,龙头带头砸盘 |
| 技术面 | 未破位 | **破位 + 基本面拐点共振** |
| **操作** | **加仓上车**(牛市主升浪大跌多为洗盘) | **保命下车**(熊市/高位横盘后大跌) |

## 12. 逃顶技巧与顶部形态

### 12.1 顶部信号清单
| 信号 | 触发条件 | 操作 |
|---|---|---|
| **天量天价** | 历史最大成交量 + 价格见顶 + 冲高回落 | **卖出** |
| **放量滞涨** | 放量但价格不涨 + 主力出货 | **卖出** |
| **高位放量大阴线/断头铡刀** | 高位放量长阴 | **见顶卖出** |
| **MACD 顶背离** | 股价新高,MACD 未新高 | **止盈/减仓,不追高** |
| **MACD 零轴下死叉** | MACD 在零轴下方死叉 + 放量下跌 | **加速杀跌,卖出** |
| **RSI > 80 回落** | 极端超买后回落 | **顶部临近,卖出** |
| **布林带触上轨收缩** | 触及上轨 + 收口 | **顶部可能** |
| **射击之星** | 高位长上影小实体 + 次日低开/放量下跌 | **卖出** |
| **黄昏之星/吞没阴线** | 高位反转 K 线组合 | **卖出** |
| **双顶(M头)/头肩顶** | 两次冲高失败/头肩顶 + 跌破颈线 + 放量加速下跌 | **卖出** |
| **跌破 60日均线/长期趋势线** | 反抽无力 | **趋势反转卖出** |
| **上涨角度过陡** | 角度 > 60° + 短期涨幅 > 50% | **高台跳水风险,卖出** |
| **尾盘急拉 > 3%** | 下午 2:50 急拉 | **次日低开概率高,谨慎** |
| **换手率 > 25% + 高位大阴线** | 主力出货 | **见顶卖出** |
| **换手率 > 70%** | 主力疯狂出逃 | **立即卖出** |

### 12.2 逃顶核心要点
- 掌握基本量价关系、均线意义。
- 顶部从来不是看到第一根大阴线才开始,主力在投资者兴奋时已悄悄开始出货。
- 多重验证:形态 + 指标 + 成交量 + 大盘环境综合判断。

## 13. 道氏理论要点

### 13.1 三种趋势
| 趋势类型 | 持续时间 | 含义 | 操作 |
|---|---|---|---|
| **主要趋势(Primary)** | 一年或更长 | 牛市/熊市大方向,最重要 | **顺应主要趋势操作** |
| **次要趋势(Secondary)** | 三周至三个月 | 对主要趋势的回调/反弹,幅度为主要趋势的 1/3 ~ 2/3 | 牛市=上涨中调整;熊市=下跌中反弹 |
| **短期趋势(Minor)** | 数天至数周 | 日内/短期波动,噪音 | 避免受其干扰 |

- **牛市主要趋势**:高点抬高 + 低点抬高。
- **熊市主要趋势**:高点降低 + 低点降低。

### 13.2 牛市三阶段
1. **吸筹阶段(Accumulation)**:市场极度悲观,价格低位;聪明资金低吸建仓,成交量逐步放大;价格缓慢上涨。
2. **拉升阶段(Markup)**:市场情绪好转,资金大量涌入;价格上涨明显,成交量持续放大。
3. **疯狂阶段(Excess)**:大众疯狂追涨;过度乐观,泡沫迹象;聪明资金开始出货,顶部将至。

### 13.3 熊市三阶段
1. **派发阶段(Distribution)**:市场极度乐观,价格高位;聪明资金高位派发;情绪转空,价格回落。
2. **下跌阶段(Decline)**:市场情绪恶化,恐慌盘涌现;价格快速下跌,成交量放大。
3. **恐慌阶段(Despair)**:极度悲观,恐慌性抛售;聪明资金抄底建仓;底部即将形成。

### 13.4 核心原则
1. **市场指数必须相互印证**:不同市场/板块走势互证才能确认趋势(大盘创新高 + 行业创新高 = 牛市确认;单一指数新高而其他未跟随 = 假突破)。
2. **成交量必须配合趋势**:
   - 上涨趋势 → 成交量逐步放大。
   - 下跌趋势 → 成交量逐步萎缩。
   - 量价背离 → 趋势可能反转(价涨量缩=诱多;价跌量缩=空头陷阱)。
3. **趋势延续直到明确反转**:趋势一旦形成将持续,直到出现明确反转信号(突破重要支撑/阻力、量价背离、重大消息面变化)。
4. **盘中波动属噪音**:日内/短期波动不代表趋势反转,不过度解读。

### 13.5 局限性
- **滞后性**:事后确认理论,趋势确认滞后于市场行为。
- **无法预测未来**:趋势跟随理论,无法预测黑天鹅。
- **短期判断不准**:适用于中长期趋势分析,短期信号可能失灵。

### 13.6 实战应用
- 通过"高点抬高、低点抬高"判断上涨;"高点降低、低点降低"判断下跌。
- 主要趋势拉升阶段 → 顺势做多;次级趋势调整阶段 → 分批低吸/修复仓位。
- 价格上涨 + 成交量放大 → 趋势有效;价格创新高 + 成量萎缩 → 谨防诱多。
- 识别反转信号:前期低/高点被突破、成交量异常、情绪极度乐观/悲观。

## 附:盈亏比与胜率(交易系统核心)

### 盈亏比 = 期望获利 / 期望亏损

### 盈亏比与最低胜率(盈亏平衡点)
| 盈亏比 | 最低胜率 |
|---|---|
| 1:1 | 50% |
| 2:1 | 33.3% |
| 3:1 | 25% |
| 4:1 | 20% |
| 5:1 | 16.7% |

### 应用原则
- **高盈亏比(≥ 3:1)** → 允许低胜率,适合趋势交易、突破交易(持仓时间较长)。
- **低盈亏比(≤ 1:1)** → 需高胜率(≥ 50%),适合超短线、日内交易。
- **避免盈亏比 < 1 的交易**(止损大于止盈,即使胜率高也可能亏损)。
- **最优策略**:提高盈亏比 + 优化胜率 + 资金管理。
- 低位起涨趋势股易获更高盈亏比(方向明确、止损明确、止盈可设远)。

## 附:集合竞价要点

### 时间安排
- 9:15-9:20:可挂单、撤单。
- 9:20-9:25:只能挂单,不能撤单。
- 9:25:撮合成交,形成开盘价。

### 高低开幅度信号
| 幅度 | 含义 | 操作 |
|---|---|---|
| 温和高开(1%-3%) | 资金关注 | 需确认盘中走势 |
| 大幅高开(> 5%) | 警惕高开低走(消息刺激) | 谨慎 |
| 温和低开(-1% ~ -3%) | 可能洗盘 | 关注回升 |
| 大幅低开(< -5%) | 可能重大利空 | 谨慎接盘 |

### 看盘要点
- 竞价量价关系反映市场初步看法。
- 异常高开/低开识别主力动向。
- 封单量大且稳定 → 强庄关注;封单不断撤单 → 诱多/诱空。
- 高开 + 大单扫货 → 真实资金进场;否则可能诱多。
- 竞价成交量远超前几日均值 → 资金抢筹。
- 结合大盘和板块热点,避免脱离市场节奏。
